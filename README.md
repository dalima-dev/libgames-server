## Description

This is a simple CRUD API (with postgresql) for a game store app.

## Installation

First turn on your database (postgresql) server.

```bash
$ npm install
$ npm install prisma
$ npx prisma db push
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```
